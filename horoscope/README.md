# Fortune teller written in Rust #

## Installation ##
git clone this repo and run `cargo build`.

## Usage ##
```
Usage: target/debug/horoscope SIGN

Options:
    -h, --help          print this help menu
```

Valid zodiac options are:

- Aries
- Taurus
- Gemini
- Cancer
- Leo
- Virgo
- Libra
- Scorpio
- Sagittarius
- Capricorn
- Aquarius
- Pisces
